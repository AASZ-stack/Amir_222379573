The program was designed to aid with the automation of Linda’s assisted living by monitoring the conditions of her house. The system starts by connecting to the wifi, then uses the DHT22 and soil moisture sensor to check environmental conditions. The sensor data is then processed and sent to ThingSpeak so that Linda can view the data in a simple web interface.

The program, like the previous task, was designed with modularity and scalability in mind. Different functions are used to handle each specific task, including connectToWifi(), soilMoistureSensor(), tempSensor() and uploadDataToThingSpeak().

The program starts by including the necessary libraries for the sensors and ThingSpeak communication. Pins and global variables are then initialised. In the setup function, the serial connection begins and the ThingSpeak and DHT sensors are initialised. The main loop starts by calling connectToWifi() to connect or reconnect to the wifi using the credentials provided in secrets.h. The sensor reading functions are then used to store the returned values in their respective variables. These values are checked using an if statement to ensure the DHT22 sensor has not returned a NaN reading before calling the uploadDataToThingSpeak() function to upload the valid readings.
