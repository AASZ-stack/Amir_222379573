This program was designed to assist Linda for when she arrives home late at night. The system allows the user to press a button to turn on two LED lights with different durations. The porch light is on for 30 seconds, while the hallway light is on for 60 seconds.

The program was designed with modularity and scalability in mind. Different functions are used to handle specific tasks, including buttonStatus(), porchLightOn(), porchLightOff(), hallWayLightOn(), and hallWayLightOff(). Each function is responsible for a single task, improving the readability. 

At the start of the program, the pins, light durations, and a variable called eventStartTime are declared. This variable is used to store the time that the button is pressed, allowing the program to determine when each LED should be turned off. In the setup() function, the pins are initialised, and the program then enters the main loop(). The loop contains multiple conditional (if) statements. The first condition checks whether the button is pressed and the lights are currently off. If both conditions are true, the program records the current time using millis(), turns on both LEDs, and sets their corresponding Boolean values to true. The second part of the loop calculates the elapsed time since the button was pressed. It then uses conditional statements to check whether each light has reached its required duration. When the time condition is met, the corresponding LED is turned off and its boolean value is set to false (to improve efficiency). Once both lights are off, the system resets the state variables, allowing the process to repeat.
